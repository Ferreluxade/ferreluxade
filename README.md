\# Ferreluxade Cookie Extractor



Ferreluxade Cookie Extractor is a Chrome extension for extracting and migrating browser cookies across browsers.



\## Features



\- Extract all cookies from the current browser profile

\- Export selected cookies only

\- Expired and near-expiry detection

\- Export formats:

  - JSON

  - Plain text

  - Netscape cookies.txt (Firefox compatible)

\- Cross-browser session portability testing



\## Use Cases



\- Cookie backup

\- Browser migration

\- Development debugging

\- Session analysis

## Installation



1\. Download the ZIP from GitHub

2\. Extract it

3\. Open the extracted folder

4\. Ensure manifest.json is visible inside the folder

5\. Open Chrome

6\. Go to chrome://extensions

7\. Enable Developer Mode

8\. Click "Load unpacked"

9\. Select the Ferreluxade folder



\## Disclaimer



Use only on accounts and browser profiles you own and control.
