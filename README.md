\# Ferreluxade Cookie Extractor



Ferreluxade Cookie Extractor is a Chrome extension for extracting and migrating browser cookies across browsers.



\## Features



\- Extract all cookies from the current browser profile

\- Export selected cookies only

\- Expired and near-expiry detection

\- Export formats:

&nbsp; - JSON

&nbsp; - Plain text

&nbsp; - Netscape cookies.txt (Firefox compatible)

\- Cross-browser session portability testing



\## Use Cases



\- Cookie backup

\- Browser migration

\- Development debugging

\- Session analysis



\## Installation (Development Mode)



1\. Open Chrome

2\. Go to chrome://extensions

3\. Enable Developer Mode

4\. Click "Load unpacked"

5\. Select the Ferreluxade folder



\## Disclaimer



Use only on accounts and browser profiles you own and control.
